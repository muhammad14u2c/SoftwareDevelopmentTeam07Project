<h1 align="center" style="bold" size="400">
    $${\color{lightgreen}What \space is \space GaterTrader? }$$ 
</h1>

<h4 align="center" size="400">
    GaterTrade aims to incorporate the main features of sites such as Adobe Stock, Pexels, and Unsplash. Each of these media distribution organizations hold their own competitive advantage. Unsplash provides free services, where Adobe Stock allows free downloads after accepting the trial and optional licensing rights upon purchase. Each of the sites provide the ability for users to input text into the search bar and find images and videos. <br><br>
    Previewing media is available across all competitor sites. Team 7â€™s project is for SFSU students and faculty, and we will provide users with the ability to contact the provider/author of the digital media they wish to acquire. This unique ability to contact a seller  guarantees customer satisfaction. This also allows users (students) to inquire and potentially bargain for reasonable pricing. <br><br> 
    Our site incorporates lazy registration and has search functionality. We allow only students and staff with emails ending in sfsu.edu or the likes to sign up for our site. Once a member, users are allow to make post, search post, and message the owner of specific posts which draw their attention. This site is no longer on the server but individuals are still able to experience the basic site and features on their local machines. <br><br>
</h4>

<h1 align="center" style="bold" size="400">
    $${\color{lightgreen}Installation }$$ 
</h1>

<h5> Create folder to download repo above</h5>

<h5> Gitbash may be of use. Visual Studio Code platform of choice. </h5>

<h5> Use Commmand <i><b>cd</b></i> to cd into working directory. Use command <i><b>git init</b></i> for new git repo. Use command <i><b>git pull </i></b> to download project</h5>

<h5> Node required. Nodemon required. Package.json include for dependencies.</h5>

<h5>Run command <i><b>npm install</b></i> for current dependencies. This creates package-lock.json</h5>

<h5> Use Commmand <i><b>cd</b></i> to cd into application. Use command <i><b>nodemon app.js to run on local machine </b></i></h5><br><br>
